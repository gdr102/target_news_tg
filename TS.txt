Сервер:
- авторизация через специально подготовленный аккаунт (далее - бот)
- обработчик ожидания новых постов
    - функция проверки текста поста на наличие ключевых слов
      при обнаружении возвращает True
        - пересылает оригинальное сообщение в чат

- обработчик получения списка каналов по команде /sources
- обработчик подписки на канал по команде /add_source
- обработчик отписки от канала по команде /remove_source

- обработчик получения списка ключевых слов из файла keywords.json по команде /keywords
- обработчик добавления ключевого слова в файл keywords.json по команде /add_keyword
- обработчик удаления ключевого слова из файла keywords.json по команде /remove_keyword

Управление:
Управление ботом будет происходить любым участником в чате по заданным командам

- /sources - список каналов на который подписан бот
- /add_source @channel - добавить источник (бот подпишется на канал)
- /remove_source @channel - удалить источник (бот отпишется от канала)

- /keywords - список ключевых слов, по которым бот отслеживает нужные нам посты
- /add_keyword "слово" - добавить ключевое слово (ключевые слова хранятся в json файле)
- /remove_keyword "слово" - удалить ключевое слово
